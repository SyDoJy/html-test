Новый курс: React Native и Expo Router

Вышел мой новый курс! В этом курсе мы изучим React Native и Expo Router написав полноценное мобильное приложение с авторизацией и нативными возможностями. Курс будет максимально практическим, ведь всю теорию мы разбираем прямо во время создания проекта применяя полученные знания на практике. Мы не используем банальные ToDo приложения, а работаем с реальным API и настоящим приложении личного кабинета ученика. Этот курс подойдёт тем, кто хочет погрузиться в мобильную разработку имея опыт frontend разработки на React.

Курс максимально использует все возможности новой платформы обучения:

  • Конспекты к каждой лекции
  • Материалы и полезные ссылки
  • Скачиваемые презентации по курсу
  • Огромный проект с наставником, где будет реализовано приложения по заказу вкусного и горячего кофе

После прохождения курса вы сможете:

  • Создавать приложения на React Native
  • Стилизовать элементы и работать с анимацией
  • Использовать Expo Router
  • Создавать компоненты и layout экранов
  • Отлаживать приложения
  • Использовать state manager
  • Работать с персистентностью данных
  • Работать с авторизацией
  • Использовать нативные API
  • Адаптировать приложения под разные устройства
  • Работать с уведомлениями
  • Собирать приложения для production

(16 часов лекций, 12 упражнений, 14 тестов)
